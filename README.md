## Highlight Projects

### QA Sentinel — Agentic QA Automation System
Multi-agent QA workflow with Orchestrator, Planner, Test Generator, and Global Validator. Built using ADK loops, memory, evaluators, and export server.

### Mirror AI — Emotional Journaling App
Uses HuggingFace emotion-classification and MNLI models, Supabase authentication, and journaling pipeline. Real-time tagging, memory-based insights, and user sentiment history.

### Capstone Multi-Agent Pipeline
Three LLMRAP agents, three loop agents, FAISS memory, and rule/evaluator layers. Designed as production-grade agentic architecture.

### DevFolio — No-Code Portfolio Generator
React + Supabase + Tailwind + shadcn/ui. Generates hosted developer portfolio websites instantly.

### Solo Leveling Productivity Game
React Native + Expo + Zustand. Character progression, quests, stats, and daily streak mechanics.
